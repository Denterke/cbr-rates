# CBR Rates API
Api provides getting data of currencies on any date

## Docker Launch
From CI folder
```
docker-compose up -d
```

## Box Launch 
```
php artisan serve
```

## Docs
Api documentation: http://localhost/docs <br>
Cbr documentation: http://www.cbr.ru/development/SXML/
Cbr currency codes: http://www.cbr.ru/scripts/XML_val.asp?d=0
